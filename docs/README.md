## Créditos

Esta documentação é baseada no trabalho original de **Edward Martin**, criador do IntMUD.

O IntMUD é uma obra impressionante de engenharia de software que inclui:
- Um interpretador completo para uma linguagem de programação própria
- Suporte a múltiplos protocolos de rede
- Um sistema completo para criação de MUDs
- Documentação extensa e tutoriais detalhados

**Nossos sinceros agradecimentos a Edward Martin** por criar e disponibilizar este projeto incrível.
